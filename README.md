# DrinkTracker - 饮水追踪应用

一款优雅的 iOS 饮水习惯追踪应用，帮助你保持健康的水分摄入。

## 🌊 功能特性

### 今日追踪
- **水位进度卡片**：直观的液体动画效果，实时显示饮水进度
- **智能状态反馈**：根据完成度显示不同状态（需要补水/继续加油/表现不错/目标达成）
- **励志语录**：根据饮水量动态显示鼓励语
- **详细记录**：查看当天每次的饮水记录，包含时间、类型和容量

### 历史记录
- **月度热力图**：可视化展示每月饮水习惯，颜色深浅反映饮水量
- **日期详情**：点击任意日期查看详细记录
- **趋势分析**：月度趋势条，了解长期饮水模式
- **统计数据**：连续天数、年度总量、日均饮水量等

### 个性化设置
- **自定义目标**：设置每日饮水杯数目标（1-20 杯）
- **提醒功能**：可选饮水提醒通知
- **数据管理**：支持重置所有数据

## 🎨 设计特色

### 深海蓝主题配色
- 主色调：深海蓝 `#1a1a2e`
- 背景色：奶油白 `#faf7f2`
- 渐变色：白色到浅蓝的柔和过渡

### 视觉效果
- **环境光晕**：页面边角装饰性渐变光晕
- **玻璃拟态**：半透明卡片配合精细边框
- **流体动画**：水位上升动画带波浪效果
- **响应式交互**：按钮按压、数字弹出等细腻反馈

## 🏗️ 技术架构

### 数据层
- **CoreData**：本地数据持久化
- **DrinkStore**：统一数据操作接口
- **实体模型**：
  - `DrinkEntry`：饮水记录
  - `DrinkType`：饮品类型
  - `UserSettings`：用户设置

### 架构模式
- **MVVM**：清晰的视图模型分离
- **SwiftUI**：声明式 UI 框架
- **环境注入**：通过 `@Environment` 传递数据上下文

## 📱 使用方式

### 底部导航
应用采用自定义底部导航栏，包含三个主要功能：
1. **今日** - 查看和记录今日饮水
2. **历史** - 浏览历史数据和趋势
3. **设置** - 管理目标和偏好

### 添加饮水
1. 点击「添加一杯饮水」按钮
2. 在底部弹窗中选择饮品类型（水、咖啡、茶等）
3. 选择杯数
4. 确认添加
