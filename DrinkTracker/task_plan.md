# 添加饮水半弹窗 UI 重设计

## 目标
将现有的添加饮水弹窗重构为深海蓝 + 奶油白主题的 Bottom Sheet 半屏弹窗，整体风格与主界面保持一致。

## 设计要点

### 整体结构
- 从屏幕底部向上滑出的半屏弹窗（Bottom Sheet）
- 背景遮罩：深海蓝调模糊蒙层 `rgba(26,26,46,0.22)` + `backdrop-blur(4px)`
- 弹窗背景：奶油白 `rgba(250,247,242,0.98)` + `backdrop-blur(28px)`
- 顶部圆角 28px，底部贴合屏幕边缘
- 顶部细深海蓝描边
- 底部阴影：向上扩散的深蓝调柔光
- 滑入动画：0.38 秒 iOS 弹性曲线
