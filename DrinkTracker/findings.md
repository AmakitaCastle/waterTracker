# 研究发现

## 现有代码分析

### AddDrinkView.swift 当前结构
- 使用 NavigationView 包裹
- 通过 `.presentationDetents([.fraction(0.5)])` 实现半屏弹窗
- 包含：
  - 标题 "你喝了什么？"
  - DrinkTypePicker 组件选择饮品
  - 杯数选择器（圆形图标按钮 + 大数字）
  - 导航栏有取消/添加按钮

### 需要查找的组件
- DrinkTypePicker - 饮品选择器组件

### 设计规格要点

#### 颜色系统
| 元素 | 颜色 |
|------|------|
| 遮罩背景 | rgba(13,78,74,0.18) + backdrop-blur(3px) |
| 弹窗背景 | rgba(245,254,250,0.97) + backdrop-blur(24px) |
| 拖动条 | #5eead4 (50% 透明) |
| 取消按钮 | #ccfbf1 底 + #3d7a73 文字 |
| 标题 | #134e4a |
| 添加按钮渐变 | #5eead4 → #0d9488 |
| 分隔线 | 透明→薄荷绿→透明渐变 |
| 小标签 | #b2dbd7 |
| 卡片选中渐变 | #ccfbf1 → #99f6e4 |

#### 尺寸规格
- 拖动条：36x4px, 圆角 2px
- 标题栏：左右边距 22px
- 取消/添加按钮：圆角 12px
- 饮品卡片：圆角 18px, 3 列 x2 行
- 杯数卡片：圆角 20px
- 步进器按钮：36x36px, 圆角 12px

#### 动画
- 滑入：0.38 秒，cubic-bezier(0.32,0.72,0,1)
- 卡片选中：scale 1.04
